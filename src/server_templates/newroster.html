Import roster file















The file should contain the column headers id, lastname, and firstname. Additional headers that are recognized are: email, altid, twitter.
If id column is not present but email column is, then email addresses (with any common domain dropped) are normally used as unique ids.
If multi_email_id option is set, then unique ids are generated from names.

Id's are required to be unique, but names are not required to be unique (unless they are to be used for importing answers etc.)
