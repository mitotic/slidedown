Manage session {{session_name}}

  • edit: Edit session Markdown file
  • download: Download session Markdown file (and images)
  • respond: Award participation credit for lectures
  • submit: Submit test user responses (used for ending admin-paced sessions)
  • submissions: Show session submission stats, allow late submissions etc.
  • prefill: Pre-fill session with all roster users (for creating paper tests)
  • view: View session data (scores, grades, etc.)
  • import: Import responses/grades from spreadsheet.csv (from paper tests)
  • export: Export responses to spreadsheet.csv
  • delete: Delete session (IRREVERSIBLE)

Less useful options
  • qstats: Question response statistics
  • headers: List session spreadsheet columns (headers)
  • ids: List all rows (ids) in session spreadsheet
  • refresh: Refresh session cache
